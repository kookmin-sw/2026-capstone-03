# 🗺️ Cultour

환영합니다! **Cultour** 프로젝트의 공식 소개 페이지입니다.
본 프로젝트는 사용자가 특정 장소를 방문하고 스탬프를 모으는 인터랙티브 위치 기반 웹 서비스입니다.

---

## 🚀 프로젝트 소개 (Introduction)

스탬프 투어는 온·오프라인을 융합하여 사용자에게 새로운 탐험 경험을 제공합니다.
사용자는 지도를 통해 목표 위치를 확인하고, 실시간 웹캠을 통한 방문 인증을 통해 자신만의 스탬프 컬렉션을 완성할 수 있습니다.

### ✨ 주요 기능
* **소셜 로그인:** OAuth 2.0 기반의 안전하고 간편한 사용자 인증
* **인터랙티브 지도:** Kakao Maps SDK를 활용한 실시간 스탬프 위치 및 경로 제공
* **실시간 카메라 인증:** 영상을 활용한 방문 인증 및 스탬프 획득 처리
* **반응형 웹 UI:** 데스크톱 및 모바일 환경을 모두 지원하는 깔끔한 화면 구성
